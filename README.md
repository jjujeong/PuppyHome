## Description

현대 사회에서 1인 가구가 증가하고 반려동물을 기르는 가구도 증가하였다. 이로 인해 매년 10만 마리의 유기동물들이 발생하고 있지만 유기동물의 입양률은 9%에 그치는 것을 확인할 수 있었다.

유기동물을 꺼리는 이유는 ‘질병에 걸려있을 것같다’가 37.7%였다. 우리는 유기동물의 인식을 개선하고 유기동물의 입양에 도움이 되기 위해 이 앱을 기획하게 되었다.
